API Reference

Use Stripe’s no-code options or apps from our partners to get started with Stripe.

The base URL for the API is https://api.stripe.com.

  • Documentation is updated frequently.
Authentication

The Stripe API uses API keys to authenticate requests.

Test secret keys have the prefix sk_test_.

curl https://api.stripe.com/v1/charges -u sk_test_key:
  • Keep your keys secret.
Expanding Responses

Many objects allow you to expand nested objects.

This reduces the number of API calls needed.

stripe charges retrieve --expand=customer
  • Expansion is depth-limited.
Idempotent requests

The API supports idempotency for safely retrying requests.

Use an idempotency-key header to prevent duplicates.

--idempotency=TvqqU7glUbURMzQT
  • Keys expire after 24 hours.
Connected Accounts

You can issue requests on behalf of your connected accounts.

Include a Stripe-Account header containing the account ID.

--stripe-account acct_1032D82eZvKYlo2C
  • Standard accounts are supported.
